Introduction
============

UVault consists out of a collection of .NET libraries for performing user management operations in .NET applications and
runs of the following platforms:

- `Microsoft .NET 6 <https://dotnet.microsoft.com/en-us/download/dotnet/6.0>`_
- `Microsoft .NET 7 <https://dotnet.microsoft.com/en-us/download/dotnet/7.0>`_

How to contribute?
------------------

An open-source project can't be successful without contributions and UVault is no exception.
We accept pull requests no matter their size, but to smoothen the process of getting your pull request merged, this
documents contains a few steps to help you smoothen the process.
